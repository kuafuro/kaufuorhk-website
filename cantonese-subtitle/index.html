② 設定 預設已經係「廣東話口語」最佳組合

進階設定 & 點解要咁揀（秘技說明）
點解語言要揀「zh」唔揀「yue」？
Whisper 一強制 yue（Cantonese）好多時會「decoder 塌」—— 不停重複、出亂碼、成段廢咗。實測強制 zh + 用 large-v3-turbo 反而會穩定咁打出「係、喺、佢哋、咁樣、㗎、喇」呢啲原汁原味口語字。 所以呢版工具預設就係 zh + turbo。想試 yue 可以喺上面轉，覺得亂就轉返 zh。
提示詞（Initial Prompt） — 畀你喺電腦版 Buzz / faster-whisper 用。瀏覽器版 Whisper 暫時未支援 initial prompt，所以呢度淨係方便你 copy 去 desktop app。 
這是一段廣東話口語對話，請完全保留原汁原味的廣東話字詞，不要翻譯成書面語。例如：係囉、咁樣、佢哋、喺度、嘅、㗎、喇。

③ 字幕結果

私隱：你嘅錄音同字幕全部喺你部瀏覽器本地處理，唔會上傳去任何伺服器。 第一次用會由 Hugging Face 下載模型檔（turbo 約 1.6GB，之後會 cache 喺瀏覽器，第二次即刻用）。 建議用 Chrome / Edge（有 WebGPU 最快）；冇 WebGPU 會自動轉用相容模式（細啲嘅模型、慢啲）。
