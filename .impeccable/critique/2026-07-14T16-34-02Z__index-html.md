## Anti-Patterns Verdict

Reads as AI-editorial. Sits in the currently-saturated "display-serif + tracked uppercase eyebrows + hairline rules + numbered items + justified monochrome-plus-one-accent" lane. 7+ eyebrow kickers on one page; 01/02/03 numbered service markers; Fig.1 figcaption; justify on every lead. Font pairing Cormorant Garamond + Lora are both reflex-reject training defaults. This is a SHIPPED identity (identity-preservation applies) but the observation stands: no ownable device beyond a red hairline.

Deterministic scan (detect.mjs):
- index.html EXIT 2: numbered-section-markers (01/02/03) TRUE POSITIVE at lines 394/404/414; broken-image at line 430 FALSE POSITIVE (matched `<img>` inside an HTML comment / placeholder).
- login/index.html EXIT 0 clean.
- cantonese-subtitle/index.html EXIT 2: layout-transition (transition:width on 8px progress bar, low impact), em-dash-overuse (5 em-dashes).
- split-calculator/index.html EXIT 0 clean.
- Detector blind spot: homepage body copy is JS-injected from the I18N object, so em-dash-overuse UNDER-counts index.html — the rendered page is heavy with em-dashes ("workflows — we hand…", "messy data — turning…", etc.); the cadence tell is present and heavier than the scan reports.

Browser evidence (Chromium/Playwright, local static server): layout HOLDS at 1440 and 390 with ZERO horizontal overflow; hero image loads (1440×957, dimensions set, no CLS); Motion Lab + Data plates are intentional grey hatched placeholders. Low-contrast lightest text tier confirmed visually (kickers, Fig.1 caption, stat footnote). Google Fonts + Supabase SDK were proxy-blocked in the sandbox and degraded gracefully (no uncaught errors) — confirms defensive engineering; in production they load normally. NOTE: B's "sticky CTA absent" was a probe/screenshot artifact — CSS confirms #stickyCta is position:fixed bottom, display:grid at ≤840px, so it IS present on mobile (2 buttons).

## Overall Impression

Competent, more disciplined than average AI output, with real bilingual care and mature defensive engineering — but two structural problems hold it back: (1) an unfocused value proposition (three co-equal offerings; H1 duplicates the kicker), and (2) a cross-page design-system fracture where the sub-pages look like a different company. Biggest single opportunity: commit the homepage to ONE business (automation) with Motion Lab + Data reframed as proof, and unify the sub-page identity.

## What's Working

1. Design-conscious defensive engineering — .fade content stays visible if JS never loads, IntersectionObserver 2s safety-net, form 15s timeout, graceful account-service degradation, honest empty-state placeholders (no faked stock photos).
2. Genuinely adaptive contact form — selecting a service reveals only relevant fields (budget/company vs datetime), reduced-motion respected. Real cognitive-load reduction.
3. Authentic tone-consistent bilingual voice — real spoken 口語 Cantonese (not stiff 書面語), clean parallel English; accessibility fundamentals (:focus-visible, prefers-reduced-motion, safe-area-inset, thumb-zone sticky CTA) actually present.

## Priority Issues

**[P1] Cross-page design-system fracture.** Homepage = serif / red-stroke / off-white / light-only / 4px radius; login/, cantonese-subtitle/, split-calculator/ = system sans-serif / monochrome / dark-capable / pill radius. A prospect experiences two different companies; betrays the peak the homepage builds. Fix: extend homepage tokens (serif display, red hairline, off-white, shared header/footer) to sub-pages, or reconcile to one system. Command: adapt (then harden the shared token layer).

**[P1] Muddled value proposition / unfocused brand.** Three co-equal offerings; H1 repeats the kicker verbatim (zh kicker 企業流程自動化 = H1 企業流程自動化); no differentiation/proof/"why us" in the first screen. A 5-second visitor can't say what problem Kuafuor solves for them. Fix: pick automation as the spine, demote Motion Lab + Data to proof/case-study modules, rewrite H1 to a benefit. Command: distill (then clarify).

**[P1] "Grip-strength ↔ dating" research example erodes enterprise trust.** r≈0.71 "most men's right-hand grip strength positively correlated with dating frequency" (lines 621/670). Flippant/laddish as a credential; an enterprise buyer evaluating data rigor sees a viral joke. Fix: replace with a credible ownable outcome ("cut a client's month-end close from 3 days to 20 minutes"). Command: clarify.

**[P2] Contact-form feedback weak at the highest-stakes moment.** Errors sequential (one at a time), single #formMsg is role=status (polite) even for failures; no aria-invalid/aria-describedby/inline markers; success gives no reply-channel echo, timeframe, or privacy/PDPO note despite POSTing PII to Supabase from the client. Fix: inline per-field errors + aria-invalid, switch error region to role=alert, echo "we'll reply to {email}", add one-line PDPO note by submit. Command: harden (+ clarify for microcopy).

**[P3] AI-editorial saturation + reflex-reject fonts (identity-level, not mandated).** Per-section English kickers + 01/02/03 + justified serif in the saturated editorial lane; Cormorant Garamond + Lora are training-data defaults. Fix (if revisiting identity): drop one tell per section, invent one signature device, consider a non-default serif. Command: bolder (or typeset).

## Minor Observations

- H1 duplicates the kicker verbatim — wasted hero.
- Section kickers AUTOMATION/MOTION LAB/DATA ANALYTICS/CONTACT hardcoded English, untranslated in Cantonese mode (i18n inconsistency).
- Only one real image (hero); Motion Lab + Data are placeholders → flagship reads pre-launch.
- Muted text likely fails AA: text-muted 55% of #201f1d on #f3f2f2; finding-note/figcaption 11–12px neutral-600 (~3.5–4.3:1). Verify contrast.
- text-align:justify on serif leads risks rivers.
- Motion Lab entry points appear 5× (hero, dual-path, ML CTA, footer, sticky) — over-linked; dilutes contact funnel.
- No skip-to-content link.
- Anti-spam relies solely on honeypot; public insert endpoint + publishable key leans entirely on Supabase RLS (verify RLS is enforced) — no visible rate limiting.
- Single render-blocking Google Fonts stylesheet in head (display=swap mitigates); hero 290KB with dimensions set.

## Questions to Consider

1. What if the homepage committed to ONE business (automation) and reframed Motion Lab + Data as evidence — focused firm vs portfolio of side projects?
2. What if the sub-pages inherited the editorial identity — or the homepage adopted the sans system of the tools? Which one is the real Kuafuor?
3. What if the research example were a real client outcome with a number the buyer cares about?
4. What if the page removed one scaffolding tell per section and spent that budget on one ownable device?
5. What if the contact success state echoed the reply channel + a one-line PDPO note?
